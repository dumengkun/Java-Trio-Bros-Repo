if-then and if-then-else Statements in Java

In Java, control flow statements are used to define the order in which the statements in a program are executed. The if-then and if-then-else statements are fundamental control flow structures that allow you to conditionally execute a block of code based on whether a specified condition is true or false.

1. if-then statement:

The if-then statement is the most basic control flow statement. It is used to execute a block of code only if a specified condition is true. If the condition is false, the block of code inside the if-then statement is skipped, and the program continues with the next statement after the if-then block.

The syntax for an if-then statement is:

                if (condition) 
                {
                    // Code to execute if the condition is true
                }

Here, condition is a boolean expression that evaluates to either true or false. If the condition is true, the code inside the curly braces ({}) is executed.

Example:

                int a = 5;
                int b = 10;
                
                if (a < b) 
                {
                    System.out.println("a is less than b.");
                }

In this example, since a is less than b, the condition is true, and the message "a is less than b." will be printed.

2. if-then-else statement:

The if-then-else statement extends the if-then statement by providing an alternative block of code to execute when the specified condition is false. This allows you to perform different actions depending on whether the condition is true or false.

The syntax for an if-then-else statement is:

                if (condition) 
                {
                    // Code to execute if the condition is true
                } 
                else 
                {
                    // Code to execute if the condition is false
                }

Example:

                int a = 5;
                int b = 10;
                
                if (a > b) 
                {
                    System.out.println("a is greater than b.");
                } 
                else 
                {
                    System.out.println("a is less than or equal to b.");
                }

In this example, since a is not greater than b, the condition is false, and the message "a is less than or equal to b." will be printed.

In conclusion, the if-then and if-then-else statements in Java are used to conditionally execute code based on the result of a boolean expression. These statements are essential for controlling the flow of a program and making decisions based on various conditions.

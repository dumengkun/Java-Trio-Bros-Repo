For Loop in Java

In Java, a for loop is a control flow statement that allows you to execute a block of code multiple times with a specified condition. The for loop consists of three main parts: the initialization, the condition, and the iteration statement. These parts are separated by semicolons within the for loop declaration.

The initialization is where a loop control variable is declared and initialized. The condition is an expression that is evaluated before each loop iteration. If the condition evaluates to true, the code block within the loop is executed; otherwise, the loop is terminated. The iteration statement updates the loop control variable after each iteration, typically incrementing or decrementing it.

Here is a basic example of a for loop in Java:


            for (int i = 0; i < 10; i++) {
                System.out.println("Iteration: " + i);
            }

In the example above, the loop control variable 'i' is initialized to 0, the condition is 'i < 10', and the iteration statement is 'i++' (which increments 'i' by 1). The loop will continue to execute as long as 'i' is less than 10. As a result, the loop will print the numbers 0 through 9.

For loops are particularly useful when you need to perform an action a specific number of times or when you need to iterate through elements in an array or a collection. They provide a concise and clear way to manage iterations and help prevent errors related to manual loop control.
